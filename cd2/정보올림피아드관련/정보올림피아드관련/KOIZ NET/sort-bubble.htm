버블 정렬(bubble sorting)

 

정렬 방법

    매 단계에서 인접한 원소들을 비교 및 교환하면서 가장 큰 키의 원소를 정렬하고자 하는 부리스트의 마지막으로 보낸다.

 

분석 ( 최악의 경우 )

    버블 정렬은 원소들이 키에 의해 역순으로 정렬되어 있는 경우에 가장 많은 비교가 필요하며, 이 때 비교의 횟수는

(n-1) + (n-2) + ..... + 2 + 1 = ( n ( n-1) ) /2

문제

[ 1 4 2 8 7 ] 에 대해 버블 정렬을 수행 했을 때, 원소들의 위치가 어떻게 변하는지 생각해보아라.

(1) C/C++

 void bubble(int a[], int n)
{
        int i, j, temp;
        for(i = 0; i < n-1; ++i)
            for(j = n-1; j > i ; --j)
                if(a[j-1] > a[j])
                {
                temp = a[j-1];
                a[j-1] = a[j];
                a[j] =temp;
                }
}
