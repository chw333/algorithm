평면상에 세 점이 있을 때 그 점들이 어떤 방향을 이루고 있는지를 검사하는 것은 기하 알고리즘의 기초가 되기 때문에 매우 중요한 내용입니다. a, b, c 세 점이 시계방향으로 존재하는지, 반시계방향으로 존재하는지, 그것도 아니면 일직선 상에 존재하는 지를 검사하는 방법에 대해서 알아보도록 하겠습니다.

이것은 벡터의 외적을 이용해서 구해야 합니다. 수학적인 내용에 대해서는 저도 잘 모르므로 그냥 방법에 대해서만 살펴 보겠습니다.

a.x * b.y + b.x * c.y + c.x * a.y - a.x * c.y - b.x * a.y - c.x * b.y의 값을 구하면 세 점 a, b, c가 이루는 삼각형의 넓이*2가 구해집니다. 이 삼각형의 넓이가 양수라면 a, b, c는 반시계방향이며(즉 선분 a-b를 중심으로 점 c가 선분의 왼쪽에 있다는 것을 의미) 넓이가 음수라면 a, b, c는 시계방향이 됩니다. 만약 넓이가 0이라면 세 점이 삼각형을 이루지 않고 한 직선상에 존재한다는 뜻이 되겠죠.

밑의 left함수를 삼각형의 넓이를 구하는 함수로 고치기 위해서는 abs(t)/2.0을 넓이로 되돌리시면 됩니다.

type
  Tpoint = record
    x, y : longint;
  end;

function left (a, b, c : Tpoint) : boolean;
var
  t : longint;
begin
  t := a.x * b.y + b.x * c.y + c.x * a.y
       - a.x * c.y - b.x * a.y - c.x * b.y;

  if t > 0 then left := true
           else left := false;
end;

var
  a, b, c : Tpoint;
begin
  a.x := 0;
  a.y := 0;
  b.x := 2;
  b.y := 5;
  c.x := 4;
  c.y := 3;

  writeln (left (a, b, c));
end.
