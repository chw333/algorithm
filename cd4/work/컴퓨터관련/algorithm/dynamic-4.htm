Longest Common Subsequence(LCS)

  주어지는 두개의 문자열에서, 공통된 가장 긴 문자열을 찾는 문제이다. 문자열 X={x1,x2,...,xn}이 주어졌다고 하자. 이 때, 다음과 같은 조건을 만족하는 문자열 Z={z1,z2,...,zn}을 문자열 X의 부 문자열이라 한다.

  조건 : 문자열 X의 인덱스 {i1,i2,...,in)의 각각에 대해 , x ij=z j를 만족한다.

         (j=1,2,...k)

  예로서, Z={B,C,D,B}는 X={A,B,C,B,D,A,B}의 부 문자열이 된다.

 

  두개의 문자열 A와 B가 주어지고, 문자열 Z가 문자열 A와 B에 공통되는 부문자열일 경우, 문자열 Z를 공통 부 문자열(Common Subsequence)이라 한다. 대개의 경우 두 문자열 사이에는 여러 개의 공통 부 문자열이 존재하는 것이 가능하며, 이러한 여러개의 공통 부 문자열 중 가장 긴 부문자열을 찾는 것이 여기서 제시되는 LCS문제이다.

 

  LCS문제를 해결하는 가장 간단한 방법은 두개의 문자열 각각으로 부터, 모든 부 문자열을 추출하고, 이 것을 비교하여 가장 긴 부 문자열을 찾는 것이다. 그러나 이 것은 문자열의 길이가 길어질 경우 컴퓨터로 해결하기에 너무 많은 시간을 요구하는 알고리즘이기 때문에 바람직하지 않다.

 

 

  LCS문제에 동적프로그래밍 기법을 적용시키기 위해서, 먼저 해결해야 할 문제를 더 작은 문제의 집합으로 나누는 것이다. 이 것을 위해 다음의 규칙을 정해야 한다. 두 개의 문자열 X={x1,x2,...,xn}과 Y=(y1,y2,...,y}, 그리고 Z={z1,z2,..,zk}가 두 문자열의 LCS라 하자. 규칙은 다음과 같다.

 

  1. Xn = Yn이면, Zk=Xn=Ym이고 Z k-1은 문자열 Xn-1과 Ym-1의 LCS가 된다.

  2. Xn != Ym일 때, Zk != Xn은 Z가 X n-1과 Y의 LCS임을 암시한다.

  3. Xn != Ym일 때, Zk != Ym은 Z가 Xn 과 Ym-1의 LCS임을 암시한다.

 

  위의 규칙을 이용하여, 두 문자열의 각 길이에 대한 LCS의 길이를 구하는 재귀식은 다음과 같이 나타낼 수있다.

 

  c[i,j] =   0  if i=0 or j=0  c[i-1,j-1]+1  if i,j>0 and Xi = Yi  max(c[i,j-1],c[i-1,j])  if i,j>0 and Xi != Yi
