선택 정렬(Selection Sorting)

가장 무식한 정렬 방법 중 하나지요?

(뭐, 때에 따라서는 O(n^2) 알고리즘이 좋을때도 있습니다.)

이 알고리즘은 배열에서 가장 적은 요소를 찾아서 첫번째 위치에 있는 요소와 교환하고

그후 두번째로 가장 적은 요소를 찾고, 두번 째 위치에 있는 요소와 교환을 하는겁니다. 보통 우리들이 실생활에서도 써먹는 방식이죠?

데이터를 예로들어 볼까요?

 

9 3 2 5 7         <-------- 처음데이터
2 3 9 5 7         <-------- 첫 교환      
2 3 5 9 7         <-------- 둘 교환      
2 3 5 7 9         <-------- Final 교환!!

 

어떻게 되시는지 아시겠죠?

간단한 C 코드입니다

void selection_sort()
{
   int i, j, min;
   
   for ( i = 1 ; i < N ; i++ ) {
       min = i;
       for ( j = i + 1 ; j <= N ; j++ )
               if ( data[j] < data[min] )
                       min = j;
       swap( min, i );
   }
}

 

data배열은 값이 들어있는 배열이고 swap()함수는 두 값을 교환하는거...
