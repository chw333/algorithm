행렬 곱셈(Matrix Multiplication)

  앞에서 살펴본 다이나믹 프로그래밍 기법을 이용하여 행렬 곱을 계산할 때, 계산량을 줄일 수 있는 방법을 생각해보자. p X q인 행렬 M p X q와 q X r인 행렬 M q X r의 곱은 p * q * r 호의 곱셈이 필요하다. 다음과 같은 행렬 곱을 해야 할 경우 어떤 방법으로 곱셈을 수행해 좋은지 생각해 보고, 이를 다이나믹 프로그래밍으로 해결하는 프로그램을 작성해보자.
