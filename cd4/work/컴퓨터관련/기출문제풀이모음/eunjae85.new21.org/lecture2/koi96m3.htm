이 문제를 푸는 방법으로는 여러 가지가 있겠지만, 여기서는 비교적 간단하고 버그 없는 플레인 스위핑(plane sweeping... 철자 맞나? -_-;)을 이용한 풀이를 살펴보도록 하겠습니다. 이 플레인 스위핑은 수많은 기하 문제에 사용할 수 있으니 꼭 알아두시는 것이 좋을 것입니다.

일단 사각형들을 입력 받으면, 모든 사각형들의 y좌표를 뽑아 냅니다. 그렇게 해서 얻은 y좌표를 이용해 사각형들을 쪼개 볼 수 있습니다(실제로 쪼개는 것은 아님)

그림 1

세 개의 사각형을 y좌표를 이용해 5개의 구간으로 나눴습니다. 이렇게 나눈 다음에 레이저(?)를 이용해서 왼쪽부터 사각형들을 훑어냅니다. 위의 그림에서는 빨간색 선분이 레이저입니다.

각각의 구간에는 배열을 이용해 숫자를 써 넣게 됩니다. 어떤 구간에 들어가는 숫자는 레이져가 가르키고 있는 부분까지 sweeping(쓸어내기?!)을 했을 때 그 구간에 겹친 사각형의 개수를 나타냅니다. 그 배열에는 다음과 같은 방법으로 값을 넣습니다.

그림 2

레이저가 오른쪽으로 훑으면서 한 사각형을 두 번 만나게 됩니다. 한번은 사각형의 왼쪽 변, 한번은 사각형의 오른쪽 변을 만나게 됩니다. 왼쪽 변을 만났을 때는 그 변에 포함되는 구간들에 1씩 더해주고 오른쪽 변을 만났을 때는 반대로 1을 빼줍니다.

그림 1에서 레이저의 위치와 배열에 씌여진 값에 주목해 주세요.

레이저는 사각형들의 모든 x좌표를 뽑아 정렬한 다음 왼쪽부터 훑어 나가게 됩니다. 만약 5개의 구간에 씌여진 값이,

0 1 2 0 1 이고 레이저가 x좌표로 10만큼 이동했다면 레이저가 이동한 구간의 넓이는

(2번 구간이 길이 + 3번 구간의 길이 + 5번 구간의 길이) * 10 이 됩니다.

배열에 2가 씌여져 있다는 것은 그 구간에 사각형이 두 번 겹친다는 말인데 어짜피 넓이를 계산할 때에는 한번 겹치는 것과 상관없기 때문에 위의 식에서는 한번만 더해줍니다.

이렇게 하여 레이저가 이동한 모든 구간의 넓이를 구하면 됩니다. 이것이 플레인 스위핑을 이용한 이 문제의 간단한 풀이었습니다. 플레인 스위핑은 이렇듯, 2차원의 문제를 1차원의 문제(구간)로 줄여줘서 문제를 한결 편하게 풀 수 있도록 해 줍니다.

사각형들은 전부 쪼갠 다음 더하는 무식한(?) 방법보다 훨씬 세련미 있죠.

아래는 저희 학교 선배이신 박준혁님이 수고해 주신 소스입니다.

program area;

const
     infile             =       'input.txt';
     outfile            =       'output.txt';

var
   n                    :       integer;
   nn                   :       integer;
   yb, yt               :       array[ 1..100 ] of real;
   x                    :       array[ 1..200 ] of real;
   y                    :       array[ 1..200 ] of integer;
   mode                 :       array[ 1..200 ] of boolean;
   laser                :       array[ 1..200 ] of integer;
   lxy                  :       array[ 1..200 ] of real;
   sum                  :       real;

procedure input;

var
   inf                  :       text;
   i                    :       integer;
   x1, y1, ww, hh       :       real;

begin
     assign( inf, infile );
     reset( inf );

     read( inf, n );

     nn := n * 2;
     for i := 1 to n do
     begin
          read( inf, x1, y1, ww, hh );
          x[ i * 2 - 1 ] := x1;
          x[ i * 2 ] := x1 + ww;
          y[ i * 2 - 1 ] := i;
          y[ i * 2 ] := i;
          mode[ i * 2 - 1 ] := true;
          mode[ i * 2 ] := false;
          yb[ i ] := y1;
          yt[ i ] := y1 + hh;
          lxy[ i * 2 - 1 ] := y1;
          lxy[ i * 2 ] := y1 + hh;
     end;

     close( inf );
end;

procedure swapr( var a, b : real );

var
   p            :       real;

begin
     p := a;
     a := b;
     b := p;
end;

procedure swap( var a, b : integer );

var
   p            :       integer;

begin
     p := a;
     a := b;
     b := p;
end;

procedure swapm( var a, b : boolean );

var
   p            :       boolean;

begin
     p := a;
     a := b;
     b := p;
end;

function fy( a : real ) : integer;

var
   i            :       integer;

begin
     for i := 1 to nn do
         if a = lxy[ i ] then fy := i;
end;

procedure solve;

var
   i, j         :       integer;
   before       :       real;
   count        :       real;

begin
     for i := 1 to nn do
         for j := i + 1 to nn do
             if lxy[ i ] > lxy[ j ] then swapr( lxy[ i ], lxy[ j ] );

     for i := 1 to nn do
         for j := i + 1 to nn do
              if ( x[ i ] > x[ j ] ) then
              begin
                   swapr( x[ i ], x[ j ] );
                   swap( y[ i ], y[ j ] );
                   swapm( mode[ i ], mode[ j ] );
              end;

     before := x[ 1 ];

     for i := 1 to nn do
         laser[ i ] := 0;

     sum := 0;

     for i := 1 to nn do
     begin
          count := 0;
          for j := 1 to nn do
              if laser[ j ] > 0 then count := count + lxy[ j + 1 ] - lxy[ j ];

          sum := sum + count * ( x[ i ] - before );

          for j := fy( yb[ y[ i ] ] ) to fy( yt[ y[ i ] ] ) - 1 do
              if mode[ i ] then inc( laser[ j ] )
                           else deC( laser[ j ] );

          before := x[ i ];
     end;
end;

procedure output;

var
   ouf          :       text;

begin
     assign( ouf, outfile );
     rewrite( ouf );

     writeln( ouf, sum : 2 : 2 );

     close( ouf );
end;

begin
     input;
     solve;
     output;
end.
