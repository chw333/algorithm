[학습문제7] 

   합이 되어 얼마가 되는 수를 찾아보자.

 

[문7-1] 두 자연수의 합이 10이 되는 모든 경우를 출력하는 프로그램을 작성하시오.

 

▷처리조건

1. 더 해지는 두 수는 자연수여야 한다.

2. 사용되는 두 수가 같은 경우는 출력하지 않는다.

   즉, 1+9=10과 9+1=10은 같은 경우로 보고 둘 중에 하나만 출력한다.

▷출력결과

1 + 9 = 10

2 + 8 = 10

3 + 7 = 10

4 + 6 = 10

5 + 5 = 10

 

▷소스코드

#include <iostream.h>

void main()

{

    int i;

    for(i=1; i<=10/2; i++) {

        cout << i << " + " << (10-i) << " = 10";

        cout << "\n";

    }

}

 

 

설명

 

1. 더해서 10이 되는 수를 구하기 위하여 1에 x를 더하면 10이 되는 x를 생각하면 x는 10-1임을 알 수 있다.

2. 따라서 출력되는 수로 i와 10-i를 출력한다.

3. 이 때, 같은 수가 반복되면 안되므로 i가 1부터 10/2까지 수만큼만 반복한다.

 

 

 

[문7-2] 자연수 N을 키보드로부터 입력받아, 두 자연수의 합이 N이 되는 모든 경우를 출력하는 프로그램을 작성하시오.

 

▷처리조건

1. 더 해지는 두 수는 자연수여야 한다.

2. 사용되는 두 수가 같은 경우는 출력하지 않는다.

   즉, 1+9=10과 9+1=10은 같은 경우로 보고 둘 중에 하나만 출력한다.

 

▷입력화면

N=? 10

▷출력결과

1 + 9 = 10

2 + 8 = 10

3 + 7 = 10

4 + 6 = 10

5 + 5 = 10

 

▷입력화면

N=? 13

▷출력결과

1+12=13

2+11=13

3+10=13

4+9=13

5+8=13

6+7=13

 

▷소스코드

#include <iostream.h>

void main()

{

    int i, n;

    cin >> n;

    for(i=1; i<=n/2; i++) {

        cout << i << "+" << (n-i) << "=" << n;

        cout << "\n";

    }

}

 

 

설명

 

1. 위 문제와 다른 부분을 찾아보면, 어떻게 해야할지를 이해할 수 있다.

2. 다른 부분은 n값을 입력받았으며, 그 값에 따라서 for문의 횟수가 변하고, 출력되는 숫자가 변한다는 것이다.

 

 

 

[문7-3] 세 자연수의 합이 20이 되는 모든 경우를 출력하는 프로그램을 작성하시오.

 

▷처리조건

1. 더 해지는 세 수는 자연수여야 한다.

2. 중복되는 경우가 없어야 한다.

   즉, 1+2+17과 2+1+17, 17+1+2, 17+2+1은 같은 것으로 생각하여 그중 하나만 출력해야 한다.

3. 출력할 때, 한 줄에 두개씩 출력한다.

 

▷출력결과

1+1+18=20   1+2+17=20 

1+3+16=20   1+4+15=20 

1+5+14=20   1+6+13=20 

1+7+12=20   1+8+11=20 

1+9+10=20   2+2+16=20 

2+3+15=20   2+4+14=20 

2+5+13=20   2+6+12=20 

2+7+11=20   2+8+10=20 

2+9+9=20   3+3+14=20 

3+4+13=20   3+5+12=20 

3+6+11=20   3+7+10=20 

3+8+9=20   4+4+12=20 

4+5+11=20   4+6+10=20 

4+7+9=20   4+8+8=20 

5+5+10=20  5+6+9=20 

5+7+8=20   6+6+8=20 

6+7+7=20 

 

▷소스코드

#include <iostream.h>

void main()

{

   int i,j, count=0;

   for(i=1; i<20; i++) {

      for(j=i; j<20; j++) {

         if(20-i-j >=j) {

            cout << i << "+" << j << "+" 

                 << 20-i-j << "=20   ";

            count++;

            if(count%2==0) {

                cout << "\n";

            }

         }

      }

   }

}

 

 

설명

 

1. 더해서 20이 되기 위해서 세 번째 수를 20에서 i와 j를 뺀 값을 출력한다.

2. 그러나 단지 이것만을 하면, 중복된 결과가 나오며, 세 번째 수는 음수가 나오기도 한다.

   1+2+17=20   2+1+17=20

3. 따라서, 중복을 제거하기 위하여 두 번째 수 j는 첫 번째 수 i보다 크거나 같아야 하고, 세 번째 수 20-i-j는 두 번째 수 j보다 크거나 같아야 한다.

 

   이렇게 생각해 보자. 더해서 10되는 두 수에서 1+9=10이고, 9+1=10이다. 이때 두 번째 수가 첫 번째 수보다 큰 경우만 찾는다면, 9+1은 1이 9보다 작으므로 출력해야할 결과에서 제외된다.

4. 한 줄에 두개씩 출력하기 위해서 출력할 때 마다 수를 세고, 이 수가 짝수일 경우 한줄을 띄는 명령을 한다.

   if(count%2==0) {

       cout << "\n";

   }
