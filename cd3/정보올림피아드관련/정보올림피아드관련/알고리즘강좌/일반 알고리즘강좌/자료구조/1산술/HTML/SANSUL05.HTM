[학습문제5] 

   2차원 배열을 사용하여 봅시다. 

 

[문5-1] 3×4의 2차원 배열을 초기화하고 그 원소를 출력하는 프로그램을 작성하시오.

 

▷처리조건

데이터는 다음과 같다.

   5, 8, 2, 3

   9, 6, 7, 4

   6, 4, 6, 5

 

▷출력결과

   5, 8, 2, 3

   9, 6, 7, 4

   6, 4, 6, 5

 

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[3][4]={

        5, 8, 2, 3,

        9, 6, 7, 4,

        6, 4, 6, 5 };

    int i, j;

 

    for(i=0; i<3; i++) {

        for(j=0; j<4; j++) {

            cout << a[i][j] << " ";

        }

        cout << "\n";

    }

}

 

 

설명

 

1. 3×4배열을 초기화하기 위해 먼저 3×4변수를 선언하고, 데이터를 써준다.

   int a[3][4]={

        5, 8, 2, 3,

        9, 6, 7, 4,

        6, 4, 6, 5 };

2. 첫 번째 줄에 5, 8, 2, 3을 출력해야 하므로 for문을 사용하여 a[][칸], 즉 칸의 첨자를 0부터 3까지 증가시켜 출력한다.

   for(j=0; j<4; j++) {

       cout << a[i][j] << " ";

   }

3. 이와 같은 방법으로 3줄을 출력하기 위해서 한줄 출력하는 방법을 다시 3번 반복한다.

   for(i=0; i<3; i++) {

       for(j=0; j<4; j++) {

           cout << a[i][j] << " ";

       }

       cout << "\n";

   }

 

 

 

1 2차원 배열?

  2차원 배열이란, 1차원 배열의 묶음이 몇 줄 있는가를 말하는 것이다.

  예를 들면,

  int a[3][4];

  이라고 선언할 때, 원소의 개수가 4인 1차원 배열이 모두 3줄이 있음을 의미한다. 따라서, 2차원 배열 a는 a[줄][칸]으로 생각할 수 있다.

 

1. 2차원 배열의 선언

  형식 : 변수형 배열명[배열크기][배열크기];

  예   :  int a[3][4];

  의미 : 정수형(int)으로 a배열을 3줄×4칸, 즉 12개의 크기로 선언한다.

2. 2차원 배열의 초기화

  1차원 배열의 초기화는 다음과 같았음을 기억해 보자.

  int a[4] = { 5, 8, 2, 3 };

  2차원 배열의 초기화는 위와 같이 일차원 배열의 초기화처럼 사용하여 표기하는 것이 원칙이다.

  int a[3][4]={

      { 5, 8, 2, 3 },  //1차원 배열

      { 9, 6, 7, 4 },  //1차원 배열

      { 6, 4, 6, 5 } };

  그러나 위 문제에서

  int a[3][4]={

      5, 8, 2, 3,

      9, 6, 7, 4,

      6, 4, 6, 5 };

  로 선언한 것은 동일한 의미이지만, 간단하게 줄여서 표현한 방법이다.

 

  이와 같이 2차원 배열이 초기화 되면,

 

 

 

 

 

 

 


  표와 같이 3줄×4칸으로 들어간다.

 

  이때 1번째 줄 1번째 칸은 a[0][0]으로 표기하고(첨자가 0부터임을 기억해야 한다.), 1번째줄 2번째 칸은 a[0][1]으로, 2번째 줄 1번째 칸은 a[1][0]으로 표기한다.

 

 

 

[문5-2] 3×4의 두 2차원 배열 a[]와 b[]를 초기화하고 각각의 원소를 합하여 배열 c[]에 넣은 후 그 값을 출력하시오.

 

▷처리조건

데이터는 다음과 같다.

   a[]              b[]

5, 8, 2, 3       -3, 7, 5, 9

9, 6, 7, 4        5,-8, 4,-2

6, 4, 6, 5        7, 6, 2,-3

 

▷출력결과

2  15  7 12

14 -2 11  2

13 10  8  2

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[3][4]={

        5, 8, 2, 3,

        9, 6, 7, 4,

        6, 4, 6, 5 };

    int b[3][4]={

        -3, 7, 5, 9

        5,-8, 4,-2

        7, 6, 2,-3 };

    int c[3][4];

    int i, j;

 

    for(i=0; i<3; i++) {

        for(j=0; j<4; j++) {

            c[i][j]=a[i][j]+b[i][j];

        }

    }

    for(i=0; i<3; i++) {

        for(j=0; j<4; j++) {

            cout << c[i][j] << " ";

        }

        cout << "\n";

    }

}

 

설명

 

1. 첫줄에서는,

   c[0][0]= a[0][0]+b[0][0]

   c[0][1]= a[0][1]+b[0][1]

   c[0][2]= a[0][2]+b[0][2]

   c[0][3]= a[0][3]+b[0][3]

   이 되어야 한다. 첫줄에서 변하는 첨자는 칸의 위치에 있는 첨자가 0부터 3까지 변하므로

   for(j=0; j<4; j++) {

       c[i][j]=a[i][j]+b[i][j];

   }

   에서 j의 값이 0부터 3까지 변하도록 for문을 사용한다. 

2. 이러한 것이 첫줄, 둘째줄, 셋째줄로 반복되므로 역시 for문을 사용하여 한 줄 처리하던 내용을 3번 반복시켜며 줄의 첨자를 0부터 2까지 변하게 한다.

   for(i=0; i<3; i++) {

       for(j=0; j<4; j++) {

           c[i][j]=a[i][j]+b[i][j];

       }

   }
