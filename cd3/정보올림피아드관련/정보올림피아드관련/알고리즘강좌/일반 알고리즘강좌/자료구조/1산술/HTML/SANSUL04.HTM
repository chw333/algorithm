[문4-4] N을 입력 받아 1부터 N까지의 합을 구하는 프로그램을 작성하시오.

 

▷입력화면

num=? 10

▷출력결과

합(sum)=55

 

▷입력화면

num=? 100

▷출력결과

합(sum)=5050

 

▷입력화면

num=? 1100

합(sum)=15726

#include <iostream.h>

void main()

{

    int i, n, sum;

    cin >> n;

    sum=0;

    for(i=1; i<=n; i++) {

        sum+=i;

    }

   for()문에서 i=1로 시작하고, i<=n로 n까지 변하도록 설정한다.

   for(i=1; i<=n; i++)

[문4-5] 1에서 50까지의 수 중에서 다음과 같이 증가하는 수의 합을 구하는 프로그램을 작성하시오.

       1+2+4+7+11+16+22+29+.......

 

▷출력결과

합(sum)=175

 

▷소스코드

#include <iostream.h>

void main()

{

    int i, a, sum;

 

    i=0; a=1;

    sum=0;

    for(;;) {

        a+=i;

        i++;

        

        if(a>50) break;

        sum+=a;

    }

    cout << "합(sum)=" << sum;

}

 

 

설명

 

1. 여기서 핵심이 되는 문제는 어떻게 1, 2, 4, 7, 11, 16, 22, 29...로 증가하게 만들 것인가에 달려있다.

2. 주의깊게 위의 수를 살펴보면,

  i

0

2

3

4

5

6

7

1

4

11

16

22

29

   처럼 되어야 한다.

3. 즉, a값이 1이 되기위해서 i값 0을 더해야 하고, a값이 2가 되기위해서 a값 1에 i값 1을 더해야 하고, a값이 4가 되기위해서 a값 2에 i값 2을 더해야 한다.

4. 따라서, i값은 0 1 2 ...로 증가해야 하므로, i++를 a를 계산한 후에 넣으므로써 i는 0부터 증가하게 된다.

5. for(;;)문이 무한루프이므로 a가 50보다 크면, 더 이상 계산하면 안되고 반복문을 빠져 나와야 한다.
