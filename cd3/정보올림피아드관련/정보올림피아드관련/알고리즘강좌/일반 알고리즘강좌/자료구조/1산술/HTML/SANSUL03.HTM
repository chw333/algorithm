[학습문제3] 

   데이터를 처리하는 방법을 알아 봅시다.

 

[문3-1] 다음의 데이터를 사용하여 총점과 평균을 구하는 프로그램을 작성하시오. 평균의 값이 소수가 나오면, 소수는 출력하지 않고 몫만 출력한다.

 

▷처리조건

데이터로는 15, 87, -24, 36, 52, -18, -44,

9를 사용한다.

 

▷출력결과

총점(sum)=113

평균(avg)=14

 

1 평균이란?

1. 평균을 쉽게 생각하면 가운데 값이라고 생각할 수 있다.

   예를들어,

   4, 8의 평균은 (4+8)/2=6이고, 5, 10, 9의 평균은 (5+10+9)/3 =8이다.

2. 따라서, 평균은 총합/갯수이다.

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[8]={15, 87, -24, 36, 52, -18, -44, 9};

    int i, sum, avg;

    sum=0;

    for(i=0; i<8; i++) {

        sum+=a[i];

    }

    avg=sum/8;

    cout << "총점(sum)=" << sum << "\n";

    cout << "평균(avg)=" << avg << "\n";

}

 

 

설명

 

1. for문을 사용하여 sum에 a[]배열의 값들을 모두 더한다.

2. 평균값은 총합이 다 구해진 후에 총합/8을 하여 구한다.

3. 실제 평균값은 14.125이나, avg변수의 변수형을 int로 했으므로 소수를 없애버리고 14만 갖게 된다.

 

 

 

[문3-2] 다음의 데이터를 사용하여 총점과 평균을 구하는 프로그램을 작성하시오. 평균의 값이 소수가 나오면, 소수로 출력한다.

 

▷처리조건

데이터로는 15, 87, -24, 36, 52, -18, -44, 9를 사용한다.

▷출력결과

총점(sum)=113

평균(avg)=14

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[8]={15, 87, -24, 36, 52, -18, -44, 9};

    int i, sum;

    float avg;

    sum=0;

    for(i=0; i<8; i++) {

        sum+=a[i];

    }

    avg=(float)sum/8;

    cout << "총점(sum)=" << sum << "\n";

    cout << "평균(avg)=" << avg << "\n";

}

 

 

설명

 

1. 소수를 표현하기 위해서 avg변수를 float로 선언했음을 알아야 한다.

2. 또한, sum/8은 sum이 정수형변수이므로 이 계산 결과도 정수형으로 만들어지므로 avg변수에 넣기전에 이미 14로 된다.

   따라서, (float)sum/8로 형변환연산을 해야 한다.

 

 

1 형변환이란?

 

  변수에는 종류가 있다. int, float, char형 변수가 있는데, 그 의미를 보면 int는 정수형 변수, float는 소수의 표현이 가능한 실수형변수, char는 문자형 변수이다.

 

  경우에 따라서는, 이러한 변수의 변수형을 변경시켜야 하는데, 이를 형변환이라고 한다.

1. 형식 : (자료형)변환대상;

   avg=(float)sum/8

   에서 sum의 형이 int인데, 이를 float로 변환시킨다. 이렇게 형변환된 sum에 8을 나누면 소수까지 구해질 수 있고, 이 값을 avg에 넣는다.

 

 

 

[문3-3] 다음의 데이터 중에서 홀수번째 원소의 총점과 평균을 구하는 프로그램을 작성하시오. 평균의 값이 소수가 나오면, 소수로 출력한다.

 

▷처리조건

데이터로는 15, 87, -24, 36, 52, -18, -44, 9를 사용한다.

 

▷출력결과

총점(sum)=-1

평균(avg)=-0.25

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[8]={15, 87, -24, 36, 52,

             -18, -44, 9};

    int i, sum, count;

    float avg;

    sum=0; count=0;

    for(i=0; i<8; i++) {

        if(i%2==0) {

           sum+=a[i];

           count++;

        }

    }

    avg=(float)sum/count;

    cout << "총점(sum)=" << sum << "\n";

    cout << "평균(avg)=" << avg << "\n";

}

 

 

설명

 

1. 소수를 표현하기 위해서 avg변수를 float로 선언했다.

2. 홀수번째 원소는 1번원소(a[0]), 3번원소(a[2]), 5번원소(a[4]), 7번원소(a[6])으로 첨자가 0, 2, 4, 6이 된다.

3. 따라서 첨자%2(i%2)로 나머지가 0일 때 첨자는 0, 2, 4, 6으로 변하기 때문에 if(i%2==0)을 사용했다.

4. count++을 한 이유는 홀수번째 원소가 몇 개가 되는지를 세기 위함이다.
