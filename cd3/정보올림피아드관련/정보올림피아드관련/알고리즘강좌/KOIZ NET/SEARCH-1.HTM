선형 탐색 (Sequential Search , Linear Search)

 첫 원소부터 시작하여 키가 k인 원소를 발견하거나 리스트의 끝에 도달할 때까지 순차적으로 찾아가는 방법(주어진 엔트라는 정렬되어 있지 않다).

 

탐색 방법

         원소들이 배열에 저장되어 있는 경우

 

(1) C/C++

int LinearSearch ( int List[], int k, int end)

{

    int location=1;

    while ( location <= end )    { // 찾을 때까지

        if (List[location] == k) { // 주어진 값(k)가 찾았으면

            return location;

        } else

        Location ++;

    }

    return 0;

}

(2) Pascal

Function LinearSearch( List : array ; K,END : integer) : integer;

begin

    Location : integer;

    Location := 1;

    while Location <= END

    begin

        if (List(Location) = K)

        begin

            LinearSearch := Location;

                 Else

            Location := Location + 1;

        End;

    End;

    LinearSearch := 0;

End;

분석

    최악의 경우는 하나의 키를 찾기 위해 리스트의 끝까지 비교를 해야 한다. 따라서 최악의 경우 비교 횟수는 n
