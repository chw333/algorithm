미로찾기 프로그램을 통해 DFS를 알아볼까요?

먼저 미로찾기 프로그램은 일단 가다가 막히면 다시 돌아가는 것입니다

그리고 가는 곳이 있으면 계속 죽가는 것이죠. 이러한 성질의 가지는 것중

그래프 탐색 기법이 바로 DFS입니다.

DFS는 Depth First Search의 준말로 깊이 우선 탐색이라고 부르기도

합니다.

이 DFS 기법은 재귀 호출을 이용하는 방법으로 알고리즘 테크닉 중 백트래

킹(Back Tracking)기법에 자주 쓰이는 방법이므로 알고리즘 공부하는데

BFS(넓이 우선 탐색)가 함께 꼭 알아야 할 것입니다.

재귀호출을 이용해 간단한 DFS 알고리즘의 대강의 소스를 적어보겠습니다.


void DFS( int v ) // v : 지금 찾으려는 정점(node=vertax)
{
