[문제 2] Upperway (지상철 계획하기)

미래도시에 사는 철이의 아버지 철수씨는 미래도시의 열차사업부 수석 프로그래머이다. 그런데 미래도시는 실로 오랜만에 커다란 계획을 하나 세웠다. 바로 지상철이 도시에 다니게 하는 계획이다. 지상철역과 다음 역과의 사이는 모두 같다.
미래도시는 여러 가지 기술을 종합해 본 결과, 열차가 직선으로 뻗어갈 수는 없다는 연구가 나왔다. 따라서, 어떤 구간에서는 지상철이 경사가 지게 하여 달려야 한다.
그런데 지상철은 경사가 많이 지면 속도가 느려지기 때문에, 최대한 속도를 끌어 올려야 한다. 이때 한 역 사이의 지상철의 속도는 다음과 같이 정의된다.
 

높이를 n0, n1로 표시할 때
I) n0 > n1 : 61-(n0-n1)2
II)n0 <= n1 : 61-(n1-n0)

철수씨는 상사의 명령을 따라 지상철의 속도가 가장 빠른 경우를 구해야 한다. 철수씨를 도와주시오.

▶ 입력 형식 (upperway.in)
 

Line 1.

N
1<=N<=10000

N : 역의 갯수

Line 2. ∼
Line. n+1

Si, Ei
-30 <= Si, Ei <= 30
Si <= 0 <= Ei
Si <= Ei

Si ∼ Ei 높이 까지 역을 세울 수 있다.


▶ 출력 형식 (upperway.out)
 

Line 1.

f

나올 수 있는 가장 큰 각 역과 역 사이의 속도 평균을 소수점 세 번째 자리에서 반올림하여 소수점 두 번째 자리까지 출력한다.


▶ 입력 예
3
0 30
-30 30
-30 -30

▶ 출력 예
1815.50

▶ 입출력 해설 : 역 0은 높이 0에 건설 / 역 1은 높이 30에 건설 / 역 2는 높이 -30에 건설

▶ 시간제한 : 2초
