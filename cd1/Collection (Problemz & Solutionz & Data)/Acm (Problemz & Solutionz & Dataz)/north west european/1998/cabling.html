Dormitory cabling

Background

Finally Darmstadt's dormitories are to be connected to the Internet. Of course a large amount of cable is needed to do this. Because cable is quite expensive the Studentenwerk Darmstadt tries to find a way how to reduce the consumption of cable. Luckily a genius recently found a revolutionary way for setting up a network. With his new induction technique it is not necessary to connect the computers directly. Instead, all buildings are enclosed by one cable and data can be transferred by induction. Happy about saving a lot of money the Studentenwerk now needs a new algorithm to compute how much cable must be ordered. Therefore the length of the cable encircling a dormitory is to be computed with a precision of one meter.

The Problem

For a given set of buildings belonging to the same dorm, the length of the shortest cable that encircles all the buildings is to be computed. The buildings are all rectangular and their southern walls are exactly aligned to the x-axis of the coordinate system.

The Input

The first line of the input contains the number of dormitories. For each dormitory the first line gives the number of buildings (less than 1000). Every building is specified by a line   x  y  w  h   describing the lower left corner in integer coordinates and the width (west to east) and height (south to north) in integers. The origin of a dormitory (0, 0) is bottom left. The maximum value for x + w and y + h is 10000.

The Output

For every dormitory output the length of the cable encircling all buildings (rounded up to the next integer) in a separate line.

Sample Input

2
3
3 6 1 2
0 3 3 1
4 0 4 5
1
0 0 1 1

Sample Output

26
4
