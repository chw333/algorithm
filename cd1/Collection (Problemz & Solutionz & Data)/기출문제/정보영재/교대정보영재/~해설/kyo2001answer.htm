1. 컴퓨터의 특징 

①      대용량성 – 엄청난 하드디스크(수백권의 책을 작은 하드디스크에 저장할 수 있다.

②      정확성 – 숫자 계산이 항상 정확하다.

③      창의성 – 컴퓨터는 사람에 의해 조작되어야 하므로 창의적이지는 않다.

④      신속성 – 사람이 하기에 오랫동안 해야하는 복잡한 계산을 몇초만에 해낸다.

⑤      상호연결성 – 네트워크가 됩니다. 서로 파일, 그림, 편지 등을 주고 받죠.

2.컴퓨터가 발달한 순서에 따라 바르게 나열된 것은?

②      진공관 - 트랜지스터 - 집적회로 - 고밀도 집적회로

l         진공관은 형광등처럼 생긴 것으로 회로를 표현하기 위해 수만개(집보다 덩치가 큼)가 모여 아주 작은 논리회로를 구성함

l         트랜지스터(TR)는 손바닥만한 크기의 논리회로 역할을 하는 기기

l         집적회로(SI)는 트랜지스터 같은 많은 칩들이 모여서 하나의 손바닥만한 논리회로를 구성함

l         고밀도 집적회로(LSI)는 집적회로 보다 더 촘촘히 칩들이 붙어서 구성된 논리회로

 

3.다음은 컴퓨터 구조에 대한 설명이다. 옳지 않은 것은?

① 입력장치 - 키보드나 마우스로부터 신호를 받아들인다. 

② 출력장치 - 스캐너나 프린터로 정보를 내 보낸다.

: 스캐너는 그림이나 사진의 정보를 읽어들이는 입력장치이다.

③ 제어장치 - 여러 장치들을 통제하는 명령을 내린다.

④ 주기억장치 - 데이터를 읽어와서 잠시 보관한다.

⑤ 보조기억장치 - 나중에 다시 사용할 데이터를 보관한다.

 

4.다음은 BASIC 언어의 변수에 대한 설명이다. 옳지 않은 것은?

① 첫번째 문자는 숫자나 특수문자를 사용할 수 없다.

② PRINT, INPUT과 같은 예약어는 변수로 사용할 수 없다.

③ 문자형 변수의 끝에는 $를 붙인다.

④ 변수의 길이는 10자를 넘을 수 없다. : 변수의 길이는 C언어와 마찬가지로 길이에 상관없다.

⑤ 배정도 실수형 변수의 끝에는 #을 붙인다.

 

5.다음은 컴퓨터의 데이터의 크기를 나타내는 단위이다. 큰 순서에서 작은 순서로 바르게 나열된 것은 ?

6.다음 보기에서 가정에서 인터넷을 이용하려고 할 때 꼭 필요한 것들로 묶인 것은 ?

보기

㉠ 모뎀 ㉡ 웹 브라우저 ㉢ IPX프로토콜 ㉣ TCP/IP프로토콜 ㉤ 웹 에디터

③      ㉠, ㉡, ㉣ 

l         모뎀 : 디지털 신호(컴퓨터정보)를 아날로그(전화소리)신호로 또는 아날로그 신호를 디지털 신호로 바꾸어 주는 기기

l         웹브라우저 : 인터넷 익스플로러, 넷스케이프 네비게이터 처럼 인터넷의 글을 볼 수 있도록 해 주는 프로그램

7.다음은 윈도우 98의 기능에 관한 설명이다. 옳지 않은 것은?

① 드래그 앤 드롭(Drag and Drop) - 객체를 끌어 옮기는 기능이다.

② 플러그 앤 플레이(Plug and Play) -컴퓨터의 특정 장치를 주 기판에 꼽으면 바로 인식하여 작동하는 기능이다.

③ 그래픽 유저 인터페이스(Graphic User Interface) - 사용자에게 문자위주의 명령을 내릴 수 있도록 하여 쉽게 프로그램을 실행시킬 수 있도록 하는 기능이다. : GUI라고 약자로 쓰며 아이콘을 눌러서 명령을 실행시킬 수 있도록 하는 기능

④ 멀티태스킹(MultiTasking) - 동시에 여러 개의 프로그램을 실행시킬 수 있도록 해주는 기능이다.

⑤ 액티브 데스크탑(Active DeskTop) - 바탕화면을 사용자의 취향에 따라 고칠 수 있는 기능이다.

 

8.다음 중 가장 저장 용량이 큰 매체는?

① DVD-ROM ② CD-ROM ③ ZIP/JAZZ DISK ④ Floppy Disk ⑤ Cache Memory

l        DVD-ROM은 한편의 영화처럼 큰 동영상 파일을 담을 수 있는 디스크이다.(Digital Versatile Disc의 약자임) 약 4.7GB정도 저장할 수 있다.

l        CD-ROM은 보통 640MB, 700MB, 800MB정도의 저장용량

l        ZIP/JAZZ DISK는 100M, 1G정도의 저장 용량

l        Floppy DISK는 1.44M 정도의 저장 용량

l        Cache Memory는 1M, 512KB정도의 저장 용량

⑤ ASC - 숫자의 절대값을 구한다. : 문자의 아스키코드값을 구하는 함수

 

16. 다음 중 BASIC에서 숫자를 문자로 변환하는 함수는 ? 

①       STR : STR(10)이라고 하면 문자열 “10”을 리턴한다.

②       INSTR : INSTR() 하나의 문자열에서 찾고자 하는 문자열의 위치를 리턴한다. C언어의 strstr()함수와 비슷하다.

③       WRITE : 파일에 문장을 쓰는 함수이다.

④       CHR$ : CHR$(65)라고 하면 문자 ‘A’를 리턴한다. 아스키 코드값에 해당하는 문자를 리턴한다.

⑤       VAL : VAL(“10”)이라고 하면 숫자 10을 리턴한다.
