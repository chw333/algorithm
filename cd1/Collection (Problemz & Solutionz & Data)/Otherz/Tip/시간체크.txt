전국 대회에서 도저히 시간이 없어서 
근사해를 구해야 하는 경우 

제한 시간동안 돌리면서 최적의 답을 갱신해 나가는 풀이 방법을 
사용해야 한다. 
이와 같은 방법은 최후에 사용해야 하며, 이 방법을 남용하다 보면 
실력향상에 도움이 되지 않는다. 

다음은 시간 체크 방법이다. 
================================================================ 
#include <stdio.h> 
#include <stdlib.h> 
#include <time.h> 

void main( void ) 
{ 
   clock_t start, finish; 
   double  duration; 

   start = clock();  // 시작 시간 읽기 
   do { 

       .............. 
      처리해야 할 내용... 
       .............. 

      finish = clock(); // 처리한 다음 시간 읽기 
      duration = (double)(finish - start) / CLOCKS_PER_SEC; 
   } while (duration<마감시간); 
       // 결국 마감 시간(초단위) 까지 처리해야 할 내용을 반복함 
}
