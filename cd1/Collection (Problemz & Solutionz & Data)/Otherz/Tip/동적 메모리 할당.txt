1. 동적 메모리 할당에는 C형태와 C++형태가 있다. 

2. C형태에서는 malloc함수를 사용한다. 
   malloc함수는 항상 byte단위로 몇개를 생성할지를 알려주어야 한다. 
   예를 들면, 
   char *p=(char *)malloc(100); 
   그러나 정수형 배열이면, 
   int *p=(int *)malloc(100*sizeof(int)); 
   이렇게 된다. 

   일단, malloc앞에 (char *)나 (int *)을 붙이는 것은 생성된 메모리의 
   데이터 형이 무엇인지를 모르기 때문이다. 
   왜냐하면 앞에 말했듯이 malloc는 무조건 byte형태의 크기로 넘겨주기 
   때문이다. 따라서 int형 배열에서는 100*sizeof(int)를 해주어야, 즉 
   int형의 크기 sizeof(int)를 곱해주어야 정확한 byte크기의 계산이 된다. 

   생성된 메모리를 해제할 때는 
   free(p)를 하면된다. 

3. C++형태의 메모리 생성은 훨신 쉽다. 
   C++에서는 메모리 생성에 관한 명령어(키워드)가 새로 만들어졌다. 
   명령어는 new이다. 
   int *p=new int[100]; 
   위와 같이 int배열 100개를 동적으로 생성할 수 있다. 
   
   메모리를 해제하기 위해서는 delete p;와  같이 delete명령어를 
   사용한다. 

   C++에서는 메모리를 생성하기 위해서 
   우리가 일반적으로 변수를 선언할 때의 모양의 앞에 new만 붙이면 
   된다.
