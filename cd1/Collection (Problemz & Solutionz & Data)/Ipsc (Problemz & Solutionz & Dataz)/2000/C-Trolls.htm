2000 Problem C - Trolls
작성일: 2001/06/17
작성자: 한상진
- 숲속에 트롤이 있나요?
- 있고말고, 아주 많이.
- 하지만 전 아무것도 안보여요
- 너는 그들을 보는 방법을 몰라, 앨리스. 그들은 너희를 무서워 해서 모습을 바꿨단다
- 그들은 어떻게 생겼나요?
- 여시 내 마법의 숲이 있다. 이것은 문자열이야. 'rabbittreebugabbitreetreerabbinrabbit'. 트롤은 이것의 부분문자열이란다. 한번 찾아봐.
- 토롤은 'bug'인가요?
- 아니. 힌트를 줄께. 트롤은 'bug'보다 더 많이 있단다.
- 'tree'로 숨어 있나요?
- 아니. 'tree'보다는 적지.
- 그렇담 'rabbit'이 트롤이겠군요.
- 맞아, 그것들이지.

당신의 작업은 앨리스가 방문했던 큰 숲에서 그녀를 돕는 것이다. 숲은 문자열로 주어진다. 당신은 숲에서 트롤이라고 생각되는 부분문자열을 맞춰야 한다. 요정 (이 경우에서는 IPSC 온라인 채점 프로그램)은 당신에게 다음과 같은 답들을 줄 것이다.

OK
트롤들은 당신이 짐작한 것보다 많다
트롤들은 당신이 짐작한 것보다 적다

첫번째 답변은 당신의 짐작이 맞았으며, 문제를 해결했음을 뜻한다. 그 다음 두 답변들은 요정이 당신이 짐작한 문자열의 빈도수를 세어 트롤의 빈도수와 비교한 것이다. 빈도수는 중첩될 수 있다(주 - 부분문자열들이 서로 forest안에서 겹칠 수도 있다는 뜻인듯). 트롤과 같은 회수로 출현하는 부분문자열은 없다. 당신은 각각의 입력에 대해 10번의 추측만이 가능함을 기억하라.

입력 형식

입력 파일은 소문자로 이루어진 문자열을 담고 있다. 이 문자열은 각 라인으로 나뉘어져 있으며 78글자를 넘지 않는다. 공백은 무시되어야 한다. 주어지는 문자열이 숲이다.

출력 형식

출력 파일은 트롤이라고 짐작되는 하나의 단어를 담아야 한다.

Example
Input file:
rabbittreebugabbitreetreerabbinrab
bit

Output file:
rabbit
