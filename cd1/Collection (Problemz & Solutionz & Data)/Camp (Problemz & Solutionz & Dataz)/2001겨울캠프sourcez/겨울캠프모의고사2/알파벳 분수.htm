알파벳 분수


  알파벳 분수란 분자와 분수가 알파벳으로 이루어진 분수인데, 만약 n개의 글자로 되어 있다면 알파벳의 처음 즉, 'a'부터 n개의 글자로 이루어져 있다. 모든 n개의 글자는 반드시 한 번만 사용되어야 한다. 분자와 분모는 적어도 한 글자를 포함해야 하며 ‘/’ 기호로 구분된다. 그러므로 ABC/D, CAFE/BD는 알파벳 분수이고 CAF/ABDE, AB/D, B/, ABC는 아니다.


  어떤 알파벳 분수들은 알파벳의 순서대로 n개의 글자를 나열하고 인접한 글자 사이에 ‘/’기호를 넣은 뒤 n-1개의 괄호쌍을 이용하여 얻을 수 있다. 예를 들어서 CAF/BED는 (A/(B/((C/D)/(E/F))))로 나타낼 수 있다. 이것을 CAF/BED의 알파벳 확장이라고 한다.


A/B

B/A

A/BC

CAF/BED

ACDE/BFGH

#
예:

알파벳 분수

알파벳 확장

A/B

(A/B)

B/A

None

CA/BD

(A/(B/(C/D))) and ((A/(B/C))/D)
‘

주의: A/(B/(C/D))는 가장 처음에 있는 ‘/’에 대응하는 괄호가 없으므로 알파벳 확장이 아니다.


입력 형식: alfrac.inp

  입력 화일의 각 줄에는 알파벳 분수 하나씩이 포함되며, 이들은 모두 대문자로 되어 있다.

입력 화일의 가장 끝 줄에는 ‘#’이 있다.




출력 형식: alfrac.out

(A/B)

None

((A/B)/C)

(A/(B/((C/D)/(E/F))))

(A/(((B/C)/D)/(((E/F)/G)/H)))
  각 줄에는 입력 화일의 각 줄에 해당하는 알파벳 확장을 출력한다. 여러 가지가 있을 경우 하나만 출력하며, 없을 경우에는 ‘None'을 출력한다.
