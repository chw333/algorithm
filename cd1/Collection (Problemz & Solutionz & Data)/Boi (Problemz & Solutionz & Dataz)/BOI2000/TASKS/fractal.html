Fractal Dimension

Clement learned at school how to compute the fractal dimension of a geometric figure. For this purpose he draws a unit grid with integer coordinates. Then he draws the figure on this grid with N vertices. These N vertices define a polygon that is not necessarily convex and whose lines do not intersect.  He has to count the number K1 of unit grid squares that intersect with the figure. By intersects we mean that the square has a point in common with the interior of the figure. Then he deletes the vertical grid lines with odd x-coordinates and horizontal grid lines with odd y-coordinates and counts the number K2 of the new grid squares that intersect with the figure. The fractal dimension is computed by some formula depending on K1 and K2. Clement wants to compute the fractal dimension of the Ohrid Lake.

Task

Find K1 and K2.

Limitations

2<N<1000 is positive integer number representing the number of vertices.

Each vertex is defined by two integer numbers xi and yi representing the XY coordinates (-32000 £ xi, yi £ 32000).

Each vertex is connected with the one preceding it and the first vertex is connected with the last one by a straight line.

Time limit: 1 sec.

Executable file FRACTAL.EXE

Input file FRACTAL.IN

The first line of the input file has integer number N. In each of the next N lines there are two integer numbers xi and yi.

Output file FRACTAL.OUT

The output consists of one line with two numbers K1 and K2.

Evaluation

For each test case, 5 points will be given for correct answer for K1 and 10 points for complete solution.
