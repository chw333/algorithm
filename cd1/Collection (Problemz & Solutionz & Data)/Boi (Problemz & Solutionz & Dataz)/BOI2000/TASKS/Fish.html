Fish

Fisherman Naum has caught N fish. The weight of each fish ti is measured at the coast. Naum works for one fishing company and he has right to take home at most T grams fish. He wishes to take as few as possible number of fish, but with maximal possible weight L( £ T ).

Task

Decide which fish to take home.

Limitations

N £ 500000 is positive integer and is the number of fish.

T £ 7000 is positive integer and is the maximum weight of fish he can take home.

ti  £ 35000 is positive integer and is the weight of the i-th fish.

Time limit: 13 sec.

Executable file FISH.EXE

Input file FISH.IN

In the first line of the input file there are two integer numbers N and T. In each of the next N lines there is one integer number ti representing the weight of the i-th fish.

Output file FISH.OUT

If there is not a solution for T grams then the output should give the solution for the largest integer L £ T. The first line of the output consists of one integer K representing the number of fish to be taken home. In each of the next K lines there should be one integer number ki representing the weight of i-th fish to be taken home.
