Copy the source file to the input directory of the proper problem.
Compile the source file.
Run the batch file to test the task. test(?) <taskno>
The filename must be the same as the problem states.
You must test the task in the proper input directory.

For example:
C:\> copy c:\user\par.pas c:\boi99\test\par\input
C:\> cd\boi99\test\par\input
C:\BOI99\TEST\PAR\INPUT> tpc par
C:\BOI99\TEST\PAR\INPUT> testp 1
...
C:\BOI99\TEST\PAR\INPUT> testp 2
...
...
