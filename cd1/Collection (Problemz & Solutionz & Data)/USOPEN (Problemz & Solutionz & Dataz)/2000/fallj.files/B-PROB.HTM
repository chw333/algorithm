PROBLEM 1: Negative Number Bases [Traditional, Rob Kolstad]

Negative number bases are just like positive ones, except that every other place value is negative.

The number seven represented in base positive two is: 111 base 2. In base negative 2, however, the place values mean that 7 is represented as 11011 base -2. This is because

(1 x 16 + 1 x -8 + 0 x 4 + 1 x -2 + 1) = 7

since base 2's place values 1, -2, 4, -8, 16, etc. Note that negative number bases can represent both positive and negative integers without using a sign! So, negative number base representation does not use a sign (in this task, anyway).

Write a program to convert ordinary integers from the input to another supplied base.

INPUT FORMAT:
  • A single line with two integers: N B
     -30000 <= N <= 30000, the number to be converted, expressed in base 10 . -10 <= B <= -2 or 2 <= B <= 10, the new number base.

SAMPLE INPUT (file INPUT.TXT):

7 -2

OUTPUT FORMAT:

The output is a single line that contains the numerical representation of the number in the new number base.

SAMPLE OUTPUT (file OUTPUT.TXT):

11011

PROBLEM 2: Crossword Puzzle Words [Traditional, Rob Kolstad]

Consider a cross puzzle (where `-' represents a blank and `#' represents a black square):

- - - - #
- - # # -
- - - - -
- # # - -
# - - - -

A word must be at least two characters in length and contiguous as per traditional crossword puzzle rules. In this example, there are five `across' words and four `down' words.

Your job is to read in a crossword puzzle and compute the number of words across and down.

INPUT FORMAT:
  • Line 1: A single integer N, 3 <= N <= 100, the dimension of the square puzzle
  • Lines 2..N=1: N characters, each either `-' or `#' representing the puzzle

SAMPLE INPUT (file INPUT.TXT):

5
----#
--##-
-----
-##--
#----

OUTPUT FORMAT:

The output is a single line that contains two integers:

  • the number of `across' words
  • the number of `down' words

SAMPLE OUTPUT (file OUTPUT.TXT):

5 
4

PROBLEM 3: Cows on Parade [Charley Ashbacher; JRM 25:4 p299 '93]

Farmer John was marching 19 of his finest black Angus and white Jersey cows to market the other day when his wife, Farmeress Joanne, noticed that all 16 possible combinations of four successive black and white cows (e.g., bbbb, bbbw, bbwb, bbww, ..., wwww) were present (as contiguous cow-subsequences) as the parade passed by. Of course, some of the combinations overlapped others.

Your job is to find an ordering of N cows so that each of the possible sequences of C colors can be found as a contiguous `subsequence' of cows. Typical lengths and subsequences are: 2,5; 3,10; 4,19; 5,36 and others. Each input dataset is guaranteed to have a solution.

INPUT FORMAT:

A single line with two integers:
  • N, 1 <= N <= 32782, the number of cows on parade
  • S, 1 <= S <= 15, the length of the subsequence

SAMPLE INPUT (file INPUT.TXT):

10 3

OUTPUT FORMAT:

The output is N lines, each with a single `b' or `w'. The lines represent a possible ordering of the parading cows.

SAMPLE OUTPUT (file OUTPUT.TXT):

b
b
b
w
b
w
w
w
b
b
