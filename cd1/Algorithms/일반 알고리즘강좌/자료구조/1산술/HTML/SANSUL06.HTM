[학습문제6] 

   교집합과 합집합을 구해 봅시다. 

 

[문6-1] b[]배열의 내용을 a[]배열로 복사하여 a[]배열의 내용을 출력하시오.

 

▷처리조건

데이터는 다음과 같다.

7, 6, 3, 9, 5, 8

 

▷출력결과

7 6 3 9 5 8

 

▷소스코드

#include <iostream.h>

void main()

{

    int b[6]={ 7, -6, 3, 9, 5, 8 };

    int a[6];

    int i;

 

    for(i=0; i<6; i++) {

        a[i]=b[i];

    }

    for(i=0; i<6; i++) {

        cout << a[i] << " ";

    }

}

 

 

설명

 

1. b[]배열의 내용을 a[]배열로 복사하기 위해서는 각각의 원소를 복사하는 것을 반복한다.

   a[0]=b[0] //b[0]의 내용을 a[0]에 복사

   a[1]=b[1] //b[1]의 내용을 a[1]에 복사

   a[2]=b[2] //b[2]의 내용을 a[2]에 복사

   a[3]=b[3]

   a[4]=b[4]

   a[5]=b[5]

2. 같은 내용이 동일하게 반복될 때 for문을 사용하면 쉽게 프로그램할 수 있다.

   for(i=0; i<6; i++) {

       a[i]=b[i];

   }

   위의 프로그램에서 i는 0부터 5까지 반복한다. 그래서 6번 반복.

 

 

 

 

[문6-2] b[]배열의 내용 중에서 원소의 값이 3의 배수인 것만 a[]배열로 복사하여 a[]배열의 내용을 출력하고, 그 때 복사한 개수를 출력하시오.

 

▷처리조건

데이터는 다음과 같다.

7, 6, 3, 9, 5, 8

 

▷출력결과

6 3 9

 

▷소스코드

#include <iostream.h>

void main()

{

    int b[6]={ 7, 6, 3, 9, 5, 8 };

    int a[6];

    int i, count=0;

 

    for(i=0; i<6; i++) {

        if(b[i]%3==0) {

            a[count]=b[i];

            count++;

        }

    }

    for(i=0; i<count; i++) {

        cout << a[i] << " ";

    }

    cout << "\n";

    cout << count;

}

 

 

설명

 

1. b[]배열 중에 3의 배수인 원소를 찾기 위해서 if(b[i]%3==0)를 사용했다. %3을 하면 3으로 나눈 나머지가 나오는데 이것이 0이면 3의 배수이므로, 배수가 되는 수만 a[]배열에 넣는다.

2. 이때, 몇 개를 넣는지를 세기 위해서 count변수를 사용하여 조건을 만족할 때 하나씩 증가(count++)하여 계산한다.

 

 

 

[문6-3] 두 집합의 교집합을 구하고, 원소를 출력하는 프로그램을 작성하시오.

 

▷처리조건

데이터는 다음과 같다.

a[]= { 1, 3, 4, 2, 5 }

b[]= { 6, 8, 5, 9, 3, 15, 7 }

 

▷출력결과

교집합(cap)= { 3, 5 }

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[5]={ 1, 3, 4, 2, 5 };

    int b[7]={ 6, 8, 5, 9, 3, 15, 7 };

    int cap[7];

    int i,j, count=0;

 

    for(i=0; i<5; i++) {

        for(j=0; j<7; j++) {

            if(a[i]==b[j]) {

                cap[count]=b[j];

                count++;

            }

        }

    }

    

    cout << "교집합(cap)= { ";

    for(i=0; i<count; i++) {

        cout << cap[i] << ", ";

    }

    cout << "}";

}

 

 

설명

 

1. 교집합이란 두 집합의 원소중에 공통으로 존재하는 원소를 말한다.

   말이 너무 어렵군. 즉, a[]배열의 원소는 1 3 4 2 5이고, b[]배열의 원소가 6 8 5 9 3 15 7일때 공통으로 있는 원소는 3과 5이다.

2. 이러한 공통되는 원소를 찾기 위해서 a[]배열의 어느 원소가 b[]배열의 원소와 일치하는지를 비교하여야 한다.

   for(i=0; i<5; i++) {

       for(j=0; j<7; j++) {

           if(a[i]==b[j]) {

               cap[count]=b[j];

               count++;

           }

       }

   }

   위의 과정에서 i, j변수의 변화를 보면,

i

0

1

2

3

4

j

0~6

0~6

0~6

0~6

0~6

   i가 0일 때는 a[0]원소를 말하며 이것과 j는 0부터 6까지 변하면서 즉 b[0]부터 b[6]원소까지 변하면서 a[0]원소와 비교한다. 이때, 같은 원소가 있으면 교집합으로 생각하고 교집합 배열에 넣는다.

3. 같은 방법으로 a[]배열의 원소를 반복하여 교집합 배열 cap에 넣는다.

 

 

 

[문6-4] 두 집합의 합집합을 구하고, 원소를 출력하는  프로그램을 작성하시오.

 

▷처리조건

데이터는 다음과 같다.

a[]= { 1, 3, 4, 2, 5 }

b[]= { 6, 8, 5, 9, 3, 15, 7 }

 

▷출력결과

합집합(cup)= { 1, 3, 4, 2, 5, 6, 8, 9, 15, 7 }

 

▷소스코드

#include <iostream.h>

void main()

{

    int a[5]={ 1, 3, 4, 2, 5 };

    int b[7]={ 6, 8, 5, 9, 3, 15, 7 };

    int cup[15];

    int i,j, count=0, flag;

 

    for(i=0; i<5; i++) {

        flag=0;

        for(j=0; j<count; j++) {

            if(a[i]==cup[j]) {

                flag=1;

                break;

            }

        }

        if(flag==0) {

            cup[count]=a[i];

            count++;

        }

    }

    

    for(i=0; i<7; i++) {

        flag=0;

        for(j=0; j<count; j++) {

            if(b[i]==cup[j]) {

                flag=1;

                break;

            }

        }

        if(flag==0) {

            cup[count]=b[i];

            count++;

        }

    }

 

    cout << "합집합(cup)= { ";

    for(i=0; i<count; i++) {

        cout << cup[i] << ", ";

    }

    cout << "}";

}

 

 

설명

 

1. 합집합이란 A집합에 존재하거나, B집합에 존재하는 원소를 말한다.

2. 새로 합집합이 되는 원소인지 아닌지를 판단하기 위해서 이 새 원소가 합집합에 들어있는지 아닌지를 판단해야 한다.

   즉, flag값을 0으로 세팅하고 for를 돌린다.

   flag=0;

   for(j=0; j<count; j++) {

       if(a[i]==cup[j]) {

           flag=1;

           break;

       }

   }

3. for문을 돌려서 cup배열에 새로 넣으려는 원소 a[i]가 존재하면 flag는 1이 되고, 루프가 다 돌아도 존재하지 않으면 flag는 0이 되므로 for문이 끝나고 flag값이 0이 되면 새 원소를 cup배열에 넣는다.

   if(flag==0) {

       cup[count]=a[i];

       count++;

   }
