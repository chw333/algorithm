합병 정렬(Merge Sorting)

정렬방법

기본 개념 : 합병 정렬의 재귀호출을 사용한다.

단계 1 : 주어진 리스트 L을 부리스트 L1, L2로 나눈다.

단계 2 : 각 리스트를 재귀적으로 합병 정렬한다.

단계 3 : 정렬된 두 개의 부리스트를 병합하여 전체를 정렬한다.

 

분석

합병 정렬의 비교횟수 T(n) : mergeSort(L, 1, n)을 수행할 때의 비교횟수

T(n) = 2 * T ( n / 2 ) + c * n ( T ( n / 2 ) : 부리스트 mergeSort, c * n : 부리스트 merge)

T(n) = 2 ( 2 * T ( n / 4) + c * ( n / 2 ) ) + cn = 4T ( n / 4 ) + 2c * n

                                                              = 8T ( n / 8 ) + 3c * n

                                                              = 2 ^ k * T( n / 2 ^ k ) + k * c * n (k=log_2n)

                                                              = a * n + c * nlogn , ( a = c * n)

void merge(int a[] ,int b[],int low, int mid, int high)
{
        int i,l,m,k;
        l=low;
        m=mid+1;
        k=low;
        while(l<=mid && m<=high)
        {
                if(a[l]<=a[m])
                {
                        b[k++]=a[l++];
                }
                else
                        b[k++]=a[m++];
        }
        if(l>mid)
                for(i=m; i<=high; i++)
                        b[k++]=a[i];
        else
                for(i=l;i<=mid;i++)
                        b[k++]=a[i];
}
