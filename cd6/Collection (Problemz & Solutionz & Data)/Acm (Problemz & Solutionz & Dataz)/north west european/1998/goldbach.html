Goldbach

Background

In number theory, there exists the so called Goldbach'sche Vermutung which makes the assumption, that every even number greater than two is representable as sum of 2 prime numbers, i.e. n = 2t = p + q with t el {2, 3, ... } and p, q prime.

Although there doesn't exist a proof for this assumption, it is generally believed to be true. A different question is, if this decomposition is non-ambiguous or if an even number is decomposable in several ways.

The Problem

It is your task to write a program, that finds the number of different decompositions of a given even number n ( 2 < n <= 100000). Take into account, that plus is commutative, i.e. a+b and b+a is indeed the same decomposition.

You should first test, if the requirements are given (n even, n > 2). Also you should reject any input, which exceeds the upper limit (see above).

The Input

The input for your program consists of several integer values, for each of which you shall calculate the number of decompositions. The number m of "problems" is given in the first line which is followed by m further lines, each containing one number as input for one run.

The Output

For each input value, the output is one number, namely the number of different decompositions. If the input is not as required, the String illegal shall be printed. Every line shall only contain a number or this string and nothing else (especially no whitespace).

Sample Input

3
6
14
29

Sample Output

1
2
illegal
