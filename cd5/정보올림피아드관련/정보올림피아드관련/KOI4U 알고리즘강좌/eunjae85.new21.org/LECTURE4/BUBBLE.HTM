버블(거품) 정렬

하이텔 베이직 동호회에서 김문수(새파람)님의 자료를 허락을 맡고 갈무리 해 온 소스입니다. 원작자의 동의 없이는 다른 곳으로 옮길 수 없습니다.

 
  버블 소트 프로그램입니다.
 
  인접한 두 요소의 대소관계를 비교해서 순서가 어긋날 경우
값을 서로 교환합니다. 교환한 그 상태에서 계속해서 다음 요소와
비교가 이루어집니다.
 
  위 과정을 (n - 1) 회 반복하면 정렬이 끝납니다.
정렬 속도는 느리지만 간편하다는 장점이 있습니다.

DEFINT A-Z
DECLARE SUB bubble (arr(), n)
DECLARE SUB show (arr(), n)
 
n = 10
DIM a(n)
 
RANDOMIZE TIMER
FOR i = 1 TO n
    a(i) = INT(RND * 100) + 1
NEXT i
 
CLS
PRINT : PRINT "Before sorting:"
show a(), n
 
bubble a(), n
 
PRINT : PRINT "After sorting:"
show a(), n
 
END
 
SUB bubble (arr(), n)
    FOR i = 1 TO n - 1
        FOR j = 1 TO n - i
            IF (arr(j) > arr(j + 1)) THEN SWAP arr(j), arr(j + 1)
        NEXT j
    NEXT i
END SUB
 
SUB show (arr(), n)
    FOR i = 1 TO n
        PRINT arr(i);
    NEXT i
    PRINT
END SUB
