삽입 정렬

하이텔 베이직 동호회에서 김문수(새파람)님의 자료를 허락을 맡고 갈무리 해 온 소스입니다. 원작자의 동의 없이는 다른 곳으로 옮길 수 없습니다.

다른 언어를 쓰시는 분도 베이직 소스에 친숙하리라 생각합니다.

DEFINT A-Z
DECLARE SUB InsertSort (arr(), n)
DECLARE SUB show (arr(), n)
 
n = 10
DIM a(n)
 
RANDOMIZE TIMER
FOR i = 1 TO n
    a(i) = INT(RND * 100) + 1
NEXT i
 
CLS
PRINT : PRINT "Before sorting:"
show a(), n
 
InsertSort a(), n
 
PRINT : PRINT "After sorting:"
show a(), n
 
END
 
 
SUB InsertSort (arr(), n)
  FOR i = 2 TO n
        temp = arr(i)
        j = i - 1
        WHILE ((j > 0) AND temp < arr(j))
            arr(j + 1) = arr(j)
            j = j - 1
        WEND
        arr(j + 1) = temp
  NEXT i
END SUB
 
 

SUB show (arr(), n)
    FOR i = 1 TO n
        PRINT arr(i);
    NEXT i
    PRINT
END SUB
