선택 정렬

하이텔 베이직 동호회에서 김문수(새파람)님의 자료를 허락을 맡고 갈무리 해 온 소스입니다. 원작자의 동의 없이는 다른 곳으로 옮길 수 없습니다.

DEFINT A-Z
DECLARE SUB selection1 (arr(), n)
DECLARE SUB show (arr(), n)
 
n = 10
DIM a(n)
 
RANDOMIZE TIMER
FOR i = 1 TO n
    a(i) = INT(RND * 100) + 1
NEXT i
 
CLS
PRINT : PRINT "Before sorting:"
show a(), n
 
selection1 a(), n
 
PRINT : PRINT "After sorting:"
show a(), n
 
END
 
 
' 선택 정렬: 단순무식형
'           대소관계 비교해서 순서가 어긋나면 무조건 바꿈
'           더 이상 순서 바꿀 것이 없어도 비교는 계속됨
SUB selection1 (arr(), n)
    FOR i = 1 TO n - 1
        FOR j = i + 1 TO n
            IF (arr(i) > arr(j)) THEN SWAP arr(i), arr(j)
        NEXT j
    NEXT i
END SUB
 
 

SUB show (arr(), n)
    FOR i = 1 TO n
        PRINT arr(i);
    NEXT i
    PRINT
END SUB
